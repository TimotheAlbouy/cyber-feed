Méthode Route Paramètres et Contraintes Jeton Réponse Description
POST /api/add_feed.php Utilisateurs identifié - URL correct et complet OUI / Permet d'ajouter un feed
DELETE /api/delete_feed.php Utilisateurs identifié - id du feed à supprimer OUI / Permet de supprimer un feed
GET /api/get_feed.php Utilisateurs identifié - URL correct et complet OUI { link:... title:... description:... } Permet de récupérer un feed
GET /api/get_feeds_content.php Utilisateurs identifié OUI json_encode(["feeds" => $feedsContent]) Permet de récupérer le contenu des feeds
GET /api/get_feeds_url.php Utilisateurs identifié OUI json_encode(["feeds" => FeedUser::findByUsername($user->username)]) Permet de récupérer l'url des feeds
POST /api/login.php Saisies valides et correctes OUI json_encode(["token" => $user->token]) Permet à un visiteur de se connecter au site
POST /api/register.php Saisies valides - Nombre de caractères entre 3 et 36 pour le username OUI json_encode(["token" => $user->token]) Permet à un visiteur de s'inscrire au site
